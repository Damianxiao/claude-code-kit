## 🎯 30秒快速开始

1. **进入项目目录**
   ```bash
   cd /path/to/your/project
   ```

2. **运行设置脚本**
   ```bash
   ~/claude-code-prompt-sync/setup.sh
   ```

3. **重启Claude Code**

4. **开始使用**
   ```
   /project:ultrathink-task 帮我设计一个用户认证系统
   ```
